# 文章规则

## 目录

存放位置 + 命名规范：`_posts/<文章发布日期>-<文章英文 slug>.md`

## 元数据

```
layout: post
title: <文章标题>
date: <文章发布日期>
categories: <分类>
tags: <标签>
```

分类和标签的规则：
- 分类、标签用中文，而不是英文
- 一篇文章只能有一个分类，最好有一到三个标签
- 决定文章分类/标签时，先在已有分类/标签中寻找

存放位置 + 命名规范：`assets/images/<文章名称>/<图片序号>`。
